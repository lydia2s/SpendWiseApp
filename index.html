SpendWise

Personal Finance & Budgeting App

A personal finance and budgeting app designed to help users plan budgets and track day-to-day spending with a clean, student-friendly workflow and intelligent insights.

Project Overview

Mission

SpendWise addresses the complex challenge of personal finance management for students and professionals. Our app simplifies the entire process from budget creation to daily expense tracking, making financial management intuitive and accessible.

Innovation

By combining intelligent AI assistance, collaborative savings features, and personalized reminders, SpendWise transforms financial planning from a daunting task into an engaging experience that encourages users to make informed spending decisions.

Impact

Our solution enables users to stay within budget by build healthy spending habits and achieve their financial goals. The user-friendly design makes personal finance management approachable and sustainable for all.

Design & Architecture

Architectural Patterns

MVVM (Model-View-ViewModel): Clean separation of presentation logic from business logic with data binding for seamless UI updates and testability.

SOLID Principles: Single Responsibility, Open/Closed, Liskov Substitution, Interface Segregation, and Dependency Inversion for maintainable code architecture.

GRASP Patterns: Information Expert, Creator, Controller, Low Coupling, and High Cohesion principles for robust object-oriented design.

Tech Stack

Frontend & Backend: Java, Extensible Markup Language

Version Control: GitHub

Technologies: Firebase for real-time data saving, Android Studio for development

AI Integration: Ollama API for financial insights and feedback

User Interface Design

Dashboard

Comprehensive overview of finances and budgets

The main dashboard provides an at-a-glance view of current spending, budget status, and financial goals. Features interactive charts, quick expense entry, and budget progress indicators tailored for student finances.

Expense Tracking

Intuitive daily expense logging and categorization

Streamlined expense entry with photo capture, automatic categorization, and receipt scanning. Users can quickly log daily expenses with smart category suggestions and spending pattern analysis.

Budget Planning

Smart budgeting with real-time tracking

Create and manage multiple budgets for different spending categories. Features include spending alerts, budget recommendations, and progress visualization designed for student lifestyles.

Savings Goals

Personal and collaborative saving features

Set personal savings goals or join savings groups for shared financial objectives. Track savings progress, manage goal milestones, and stay motivated with achievement badges.

AI Assistant

Intelligent financial guidance

AI-powered assistant provides personalized budgeting advice, spending insights, and financial tips tailored for students. Natural language interface for easy financial guidance.

Design Philosophy

Student-Friendly: Clean, intuitive interface designed specifically for students and young professionals with minimal learning curve.

Consistency: Unified color scheme and typography across all screens with our signature dark green and white palette.

Accessibility: High contrast ratios, large touch targets, and support for screen readers ensuring inclusive design.

User Experience: Intuitive navigation with bottom tab bar, contextual actions, and minimal cognitive load through progressive disclosure.
