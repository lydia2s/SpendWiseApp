# SpendWiseApp
A student-friendly personal finance application that simplifies budget planning and expense tracking with intelligent insights and collaborative features.

Key Features

Smart Budget Management - Create and track multiple budgets with real-time progress monitoring
Expense Tracking - Quick expense logging with photo capture and automatic categorization
AI Assistant - Personalized financial advice and spending insights powered by Ollama API
Savings Circles - Collaborative savings groups for shared financial goals
Dashboard Analytics - Interactive charts and visualizations for spending patterns
Night Mode - Dark theme for comfortable viewing
